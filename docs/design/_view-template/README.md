# View Template

Usa esta carpeta solo como referencia para crear carpetas reales por vista dentro de `docs/design/`.

## Estructura mínima sugerida

- `screen.png`: referencia visual principal de la vista aprobada.
- `code.html`: referencia estructural exportada desde Stitch.

## Reglas

- Nombra la carpeta con el nombre real de la vista aprobada, por ejemplo `landing-home`, `login`, `dashboard-overview`.
- No copies `code.html` directo a producción.
- Si hay conflicto entre `screen.png` y `code.html`, manda `screen.png`.
