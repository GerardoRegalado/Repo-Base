# Roadmap

## Próximos pasos sugeridos

1. Definir un sistema base de secciones reutilizables para landings reales.
2. Integrar formularios comerciales con validación y proveedor de correo.
3. Agregar autenticación simple para `app/admin`.
4. Conectar signed uploads para media usando R2 o S3.
5. Añadir un editor de contenido interno o conectar un CMS.
6. Incorporar analítica, eventos y consentimiento de cookies cuando el proyecto lo requiera.
7. Ampliar testing con componentes críticos y smoke tests de rutas.
8. Preparar pipeline de deploy y checklist de handoff para clientes.

## Versión 2 ideal

- auth real
- gestión de contenido desde admin
- subida y reemplazo de media
- formularios productivos
- observabilidad
- pipeline CI/CD base
